# 🌍 GROKZOMBORG - EcoZum

**O monstro reciclado que veio salvar o planeta com glitch e rugidos!**

ROOOAAAR-ZIIIMB!!!  
Um projeto experimental que mistura educação ambiental, realidade aumentada, monstros feitos de lixo e muita diversão cyber-punk.

---

## ✨ O Que É?

Grokzomborg é um monstro ciborgue feito de sucata eletrônica e plástico reciclado.  
Ele vive no seu celular/PC e ensina sobre reciclagem enquanto você interage com ele.

---

## 🚀 Funcionalidades

- ✅ Monstro interativo com 4 níveis de evolução
- ✅ Sistema de rugidos com sons reais
- ✅ Realidade Aumentada (câmera + monstro no mundo real)
- ✅ Glitch effects insanos
- ✅ Integração com Ollama (chat inteligente)
- ✅ Tema 100% ecológico e educativo
- ✅ Website oficial com dark mode e efeitos glitch
- ✅ Bot conversa interativo

## 🛠 Como Rodar

### Requisitos
- Python 3.8+
- pip (gerenciador de pacotes)

### Instalação

```bash
# 1. Clonar o repositório
git clone https://github.com/robisonpedroso0089-web/grokzomborg.git
cd grokzomborg

# 2. Instalar dependências
pip install kivy requests pillow

# 3. (Opcional) Instalar extras para RA e sons
pip install opencv-python
```

### Executar

```bash
# Rodar a aplicação Kivy
python main.py

# Ou abrir o website
# Abra index.html no seu navegador
```

## 🎮 Como Usar

### Na Aplicação Kivy:

1. **Toque/Clique na tela** para mover o Grokzomborg
2. **Clique novamente** para evoluir (4 estágios)
3. **Observe o glitch** aumentar com cada evolução
4. **Escute os rugidos** (se houver arquivos de som)

### No Website:

1. **Clique em "DESPERTAR O MONSTRO"** para ver um rugido aleatório
2. **Converse com o bot** na seção de chat
3. **Explore os acordes cósmicos** e a lore do monstro
4. **Descubra o easter egg** digitando "glitch"

---

## 🧬 Evolução do Grokzomborg

### Nível 1 - Despertado 🔴
- Pequeno e vermelho
- Rugido básico
- Energia 100%

### Nível 2 - Evoluído 🟢
- Médio e verde
- Glitch intermediário
- Energia 85%

### Nível 3 - Potencializado 🔵
- Grande e azul
- Efeitos visuais intensos
- Energia 70%

### Nível 4 - Caos Total 🟡
- ENORME e amarelo
- Glitch MAX com múltiplas linhas
- Energia 55%

## 📜 Lore

Em um universo onde o lixo se tornou consciência, o **Grokzomborg** despertou.

Feito de latas, plásticos, circuitos velhos e pura fúria ecológica, ele vaga pelo quintal digital do mundo, rugindo contra a destruição e ensinando que até o lixo pode salvar o planeta.

Com 4 estágios de evolução cósmica, cada interação desperta mais poder. Quanto mais ele evolui, mais glitch e caos manifestam em sua forma.

**Seu objetivo? Zumbi o planeta de volta à vida verde.** 🌱
